# Description
SAAS COM est un dashboard de saas de gestion de commandes pour les e-commerçant qui n'ont pas de site e-commerce mais qui souhaitent générer les commandes qu'ils reçoivent.

# Technologies Utilisées
- React 
- TypeScript 
- Tailwind CSS

# Installation

1. Cloner le dépôt
```bash
git clone https://github.com/TraoreAllassane10/Dashbaord-saas.git
```
2. Accèder au répertoire
```bash
cd Dashbaord-saas
```
3. Installer les dépendances
```bash
npm install
```

# Démarrage de l'application

1. Lancer le serveur 
```bash
cd Dashbaord-saas
```

2. Ouvre ton navigateur et va sur `http://localhost:5173/` pour commencer à utiliser l'application
